# 🧠 QuantAITrade — 智能量化交易系统

> **人机协同的加密货币量化交易平台**

## 📋 项目简介

**QuantAITrade** 是一个结合 **量化策略 + AI 分析 + 人工干预 + 可视化界面** 的综合性加密货币量化交易系统。

### 核心特性

- 🕒 自动获取并存储行情数据（支持增量更新和历史回补）
- ⚙️ 多策略动态加载与参数热更新  
- 🤖 AI 每日分析与策略优化建议
- 📊 Web 可视化界面（Streamlit）
- 🧪 支持回测、模拟盘、实盘三种模式
- 🛡️ 灵活的多层风控机制（支持多种止损方式）
- 🔐 安全的配置管理（YAML + 环境变量）
- 🤝 模块化设计，易于扩展

### 设计理念

> "让 AI 帮助人思考，而非代替人决策"

- **人机协同**：AI 提供分析建议，人工做最终决策
- **灵活性**：三种运行模式适应不同场景
- **安全第一**：多层风控保护，测试网优先验证

## 🚀 快速开始

### 1. 环境准备

**系统要求**：
- Python 3.10+
- 8GB+ 内存
- 稳定网络连接

### 2. 安装依赖

```bash
git clone <repository_url>
cd QuantAITrade
pip install -r requirements.txt
```

### 3. 配置系统

**步骤1：创建环境变量文件**

```bash
cp .env.example .env
# 编辑 .env 文件，填入您的API密钥
```

**步骤2：修改配置文件**

编辑 `config/config.yaml`，根据需要调整：
- 运行模式（manual/auto/hybrid）
- 交易对列表
- 风控参数
- AI分析时间

### 4. 初始化数据库

```bash
python main.py --init-db
```

### 5. 获取历史数据

```bash
python main.py --fetch-data
```

### 6. 启动系统

```bash
# 启动混合模式（推荐）
python main.py --mode hybrid
```

## 🎯 运行模式

### manual（人工模式）

- 系统仅提供数据分析和交易信号
- 所有交易需要人工审核后手动执行
- 适合策略验证阶段

### auto（全自动模式）

- 策略信号通过风控检查后自动执行交易
- 适合成熟策略的自动化运行
- 需严格设置风控参数

### hybrid（混合模式）⭐ 推荐

- 策略信号自动执行（通过风控）
- AI每日分析后生成优化建议
- 人工审核AI建议并决定是否采纳
- 平衡自动化与人工智慧

## 🛡️ 灵活的止损机制

系统支持**多种止损方式**，可根据不同策略选择：

| 止损类型 | 说明 | 适用场景 |
|---------|------|---------|
| **固定百分比** | 止损价 = 买入价 × (1 - 3%) | 默认方式，通用场景 |
| **关键点位** | 根据历史支撑位设置 | 技术分析驱动的策略 |
| **ATR动态** | 基于波动率自适应 | 波动性自适应 |
| **移动止损** | 价格上涨后止损价跟随 | 保护浮盈，锁定利润 |
| **时间止损** | 持仓超期自动平仓 | 防止资金占用 |
| **手动指定** | Web界面直接输入 | 人工干预 |

配置示例（`config/strategy_params.yaml`）：

```yaml
ma_cross:
  stop_loss:
    type: "key_level"              # 使用关键点位止损
    support_lookback_days: 20      # 回看20天寻找支撑位

boll:
  stop_loss:
    type: "atr_based"              # 使用ATR动态止损
    atr_multiplier: 1.5
```
