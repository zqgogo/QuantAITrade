## 🚀 快速使用指南

### 1. 启动系统

```bash
# 初始化数据库（首次运行）
python main.py --init-db

# 获取历史数据
python main.py --fetch-data

# 启动混合模式（推荐）
python main.py --mode hybrid
```

### 2. 启动Web界面

```bash
streamlit run ui/app.py
```

访问: http://localhost:8501

### 3. 系统运行流程

**混合模式（推荐）**：
```
数据获取（每60分钟）
    ↓
策略分析（每15分钟）
    ↓
信号生成 → 风控检查 → 自动下单
    ↓
持仓监控（每30秒）→ 自动止损
    ↓
AI分析（每日09:00）→ 生成建议
    ↓
Web界面查看 → 人工审核采纳
```
