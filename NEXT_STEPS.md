## 🔍 问题排查

### 问题1：ModuleNotFoundError
**原因**：依赖未安装
**解决**：
```bash
source venv/bin/activate
pip install -r requirements.txt
```

### 问题2：配置加载失败
**原因**：缺少环境变量或配置文件
**解决**：
1. 检查 `config/config.yaml` 是否存在
2. 检查 `.env` 文件是否配置
3. 查看日志文件 `runtime/logs/system.log`

### 问题3：数据库连接失败
**原因**：数据库文件路径问题
**解决**：
1. 检查 `config/config.yaml` 中的数据库路径
2. 确保 `data/` 目录存在
3. 运行 `python main.py --init-db` 初始化
